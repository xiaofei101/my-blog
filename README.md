## 🚀 快速开始

### 本地开发

```bash
# 克隆项目
git clone https://github.com/xiaofei101/my-blog.git
cd my-blog

# 安装依赖
npm install

# 启动本地服务器
hexo server

# 访问 http://localhost:4000
```

### 创建新文章

```bash
hexo new post "文章标题"
```

### 部署

```bash
hexo generate
hexo deploy
```
